# Disaster Response Management System (DRMS)

A comprehensive system for managing and responding to disaster events.

## Features

- Real-time disaster monitoring
- Interactive map visualization
- Disaster data analysis
- Responsive web interface

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher) or yarn
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/DRMS.git
   cd DRMS
   ```

2. Install server dependencies:
   ```bash
   cd server
   npm install
   ```

3. Install client dependencies:
   ```bash
   cd ../client
   npm install
   ```

## Running the Application

1. Start the server:
   ```bash
   cd server
   npm start
   ```

2. In a new terminal, start the client:
   ```bash
   cd client
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3000`

## Project Structure

- `/client` - Frontend React application
- `/server` - Backend Node.js/Express server
- `/data` - Disaster data and resources
